# Парсер моделей NagaAI

Парсер для сбора данных о моделях AI с сайта https://naga.ac/models с автоматической конвертацией цен в рубли.

## Возможности

- ✅ Парсинг списка всех моделей с главной страницы
- ✅ Переход на страницу каждой модели для получения детальных цен
- ✅ Автоматическая конвертация цен из USD в RUB (курс: 1$ = 96₽)
- ✅ Экспорт данных в Excel (.xlsx)
- ✅ Экспорт данных в текстовый файл (.txt)

## Собираемые данные

Для каждой модели парсер собирает:
- Название модели
- Провайдер (компания-разработчик)
- Количество токенов
- Описание модели
- Цена входных токенов (в USD и RUB)
- Цена выходных токенов (в USD и RUB)
- Ссылка на страницу модели

## Установка

1. Убедитесь, что у вас установлен Node.js (версия 14 или выше)

2. Установите зависимости:
```bash
npm install
```

## Использование

Запустите парсер командой:
```bash
npm start
```

или

```bash
node parser.js
```

## Результаты

После выполнения парсинга будут созданы два файла:

1. **naga_models.xlsx** - таблица Excel со всеми данными
2. **naga_models.txt** - текстовый файл с форматированным списком моделей

## Структура Excel таблицы

| Столбец | Описание |
|---------|----------|
| Название модели | Полное название AI модели |
| Провайдер | Компания-разработчик |
| Токены | Количество обработанных токенов |
| Описание | Краткое описание модели |
| Цена входных токенов (USD) | Стоимость за 1M входных токенов в долларах |
| Цена выходных токенов (USD) | Стоимость за 1M выходных токенов в долларах |
| Цена входных токенов (RUB) | Стоимость за 1M входных токенов в рублях |
| Цена выходных токенов (RUB) | Стоимость за 1M выходных токенов в рублях |
| Ссылка | URL страницы модели |

## Зависимости

- **axios** - для HTTP запросов
- **cheerio** - для парсинга HTML
- **xlsx** - для создания Excel файлов

## Примечания

- Между запросами установлена задержка в 1 секунду для предотвращения блокировки
- Курс конвертации USD/RUB: 1$ = 96₽ (можно изменить в коде)
- Бесплатные модели отмечаются как "Free" / "Бесплатно"
